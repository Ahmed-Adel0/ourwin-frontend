# Color Replacement Map

## Inline Colors → Tailwind Classes

### Primary Accent Colors

- `bg-[#45B7BA]` → `bg-accent`
- `text-[#45B7BA]` → `text-accent`
- `border-[#45B7BA]` → `border-accent`
- `hover:bg-[#45B7BA]` → `hover:bg-accent`
- `hover:text-[#45B7BA]` → `hover:text-accent`
- `hover:border-[#45B7BA]` → `hover:border-accent`
- `bg-[#2c989c]` → `bg-accent-dark`
- `hover:bg-[#2c989c]` → `hover:bg-accent-dark`

### Dark Teal Colors

- `bg-[#0b253b]` → `bg-dark-teal`
- `bg-[#052636]` → `bg-dark-teal-lighter`
- `bg-[#0d3547]` → `bg-dark-teal-light`
- `border-[#1a2a35]` → `border-dark-teal-border`
- `bg-[#1a2a35]` → `bg-dark-teal-border`

### Cyan Colors

- `bg-[#003349]` → `bg-cyan-700`
- `bg-[#00445C]` → `bg-cyan-600`
- `bg-[#053447]` → `bg-cyan-800`
- `bg-[#1C657E]` → `bg-cyan-700`
- `hover:bg-[#247a96]` → `hover:bg-cyan-600`

### Page/Card Colors

- `bg-[#032131]` → `bg-page-bg`
- `bg-[#03141f]` → `bg-card-bg`
- `bg-[#062131]` → `bg-page-bg` (similar)
- `bg-[#031c29]` → `bg-card-bg` (similar)

### Text Colors

- `text-[#052636]` → `text-dark-teal-lighter`

### Border Colors

- `border-[#45B7BA]/20` → `border-accent/20`
- `border-[#45B7BA]/25` → `border-accent/25`
- `border-[#45B7BA]/30` → `border-accent/30`
- `border-[#45B7BA]/50` → `border-accent/50`
- `border-[#45B7BA]/60` → `border-accent/60`

### Shadow Colors

- `shadow-[#45B7BA]/30` → `shadow-accent/30`
- `shadow-[#45B7BA]/25` → `shadow-accent/25`
- `shadow-[#45B7BA]/20` → `shadow-accent/20`

### Focus Ring Colors

- `focus-visible:ring-[#45B7BA]/70` → `focus-visible:ring-accent/70`
- `focus:ring-[#45B7BA]` → `focus:ring-accent`
